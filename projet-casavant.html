Site web des projets Casavant

Le site web des Projets Casavant est un regroupement des créations de certains élèves de Secondaire 5 de l'école Casavant dans le but de conserver le résultats de tous nos projets durant l'année 2022-2023. J'ai créé ce site avec l'aide de l'éditeur de code Visual Studio Code, ce qui m'a permis de m'améliorer dans les languages HTML, CSS et Javascript. Bien sûr, tout le code est disponible sur mon GitHub (même s'il n'est pas très clair ...). Il est possible de visiter le site grâce à la platforme Vercel, je vais mettre le lien du site ci-dessous pour que vous puissiez aller y jetter un coup d'oeil.
